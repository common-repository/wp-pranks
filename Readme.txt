== Description ==

A playful plugin with several options to pull a joke/prank on your friends.  Guaranteed to work on your WordPress website regardless of theme, plugin or version of WordPress.

**Types of pranks:**

* B&W - Turns website into black and white like the old days.
* Upside Down - Turns website upside-down.
* Hide All Odd Numbered Paragraphs - Self explanatory, come on.
* Blurry - Turns website so blurry even eye-glasses won't help.
* Poop Emoji - Puts a poop emoji in upper left of the page.
* Comics Sans - Turns text to use the Comic Sans font.
* Insanely Large Text - Makes fonts larger for those who have really really bad eye-sight.

== Installation ==
1. Install the plugin.
1. Go to Settings > WP Pranks.
1. Select a prank from the dropdown menu.
1. Save changes.
